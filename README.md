# Web Development Mini Projects

## Project Description

This project contains **10 basic web development pages** created using **HTML, CSS, and JavaScript**.
Each webpage demonstrates a different concept of frontend web development such as forms, layouts, navigation, and simple interactivity.

These mini projects are designed for **learning and practice purposes** and can also be used as **college assignments or portfolio projects**.

---

## List of Projects

1. **Personal Portfolio Page**
   A webpage where a student can showcase their **projects, skills, and contact information**.

2. **Restaurant Menu Website**
   A single-page website displaying **menu categories and food prices** for a restaurant.

3. **Event Registration Form**
   A form where students can **register for a college tech fest event** by entering their details.

4. **Online Product Listing Page**
   A simple e-commerce style page showing **6–8 products with images, prices, and descriptions**.

5. **Travel Blog with Navigation**
   A blog-style webpage where a traveller shares **three travel stories** with navigation links.

6. **Quiz Webpage**
   A **5-question multiple choice quiz** where students can select answers and submit the quiz.

7. **To-Do List App**
   A webpage that allows users to **add, mark complete, and delete daily tasks**.

8. **Image Gallery with Lightbox Effect**
   A gallery showing **9 photos** where clicking an image opens it in a larger view.

9. **Simple Weather Info Panel (Static Data)**
   A weather display page showing **static weather information for three cities**.

10. **Responsive Landing Page for a Course**
    A promotional landing page for a **Web Development Bootcamp course**.

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Double-click any **.html file**.
4. The webpage will open in your web browser.
